GitHub PR Tab Group Sync

Each row maps a GitHub repo to a tab group. The group mirrors that repo's open PRs (one tab each); new PRs are added to the top, closed/merged ones are removed.

GitHub access

Needs read access to the repos' pull requests. Fine-grained token: Pull requests: Read-only (+ Contents: Read for private repos). Classic token: repo scope for private repos. Stored locally in this browser only.
