GitHub PR Tab Group Sync

Map each GitHub repo to a tab group. The group mirrors that repo's open PRs (one tab each); new PRs go to the top, closed/merged ones are removed.

GitHub tokens

Add one or more personal access tokens. A fine-grained token is scoped to a single owner/org, so add one token per owner/org you need (e.g. your personal account and each organization). Then pick the matching token on each mapping below. Tokens are stored only in this browser via chrome.storage.local.

LabelToken value

Create tokens at github.com/settings/tokens. Fine-grained: set Resource owner to the org, grant Pull requests: Read-only (+ Contents: Read for private). Classic: repo scope (and authorize SSO for the org if required).
